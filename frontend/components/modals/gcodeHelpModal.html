🔧 Aide G-code

Commandes de base:

  • G28 - Home tous les axes
  • G0 X10 Y5 - Déplacement rapide
  • G1 X10 Y5 F1000 - Déplacement linéaire
  • G90 - Mode absolu
  • G91 - Mode relatif

Commandes système:

  • M115 - Informations firmware
  • M114 - Position actuelle
  • M300 - Bip sonore
  • M999 - Reset/Redémarrage
  • M112 - Arrêt d'urgence
